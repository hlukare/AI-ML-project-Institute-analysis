I am doing the web dev part of the project both backend and frontend.

Backend is almost complete but frontend is left.

The flow goes like this:
The Project has Five Roles:
1. Super Admin
2. Region Lead
3. Unit Lead
4. Institute Lead

Hierarchy is present in them

Work done by each role:
1. Super Admin: Can perform all operations
2. Region Lead: Create Units, Assign Units to unit lead, And perform all operations of Unit lEad and Insitute head
3. Unit Lead: Create Institute, Assign Institute to Institute Head, Manage Institute, And perform all operations of Insitute head.
4. Institute Head: Manage the courses within institute, has a local server installed on the institute's system which can automatically capture frames after every specified interval and batch upload them to server, manually upload images in case of failure.
